# Figwrapper

`Figwarapper` is a script that integrates the [Figma](https://www.figma.com/) design tool (web version) generated by nativefier into an complete, fully integrated app for Linux. It uses a [Electron](https://github.com/electron/electron) wrapper generated by [nativefier](https://github.com/jiahaog/nativefier) that is further customized to behave like a proper Linux app.

## Requirement
* Internet connection.

## Install

Extract Figwrapper to a folder of your preference, then on a terminal:

```bash
> chmod u+x InstallFigma.sh
> sh ./InstallFigma.sh
```

## Uninstall

```bash
> chmod u+x InstallFigma.sh
> sh ./UninstallFigma.sh
```
